# DateTimeMorph
A C# utility class for flexible datetime manipulation and natural language parsing. Supports navigating through time periods (second to year), calculating period boundaries, and parsing human-readable date expressions like "next Monday," "3 days ago," or "in 2 weeks."

# Features
Navigate through time periods (second, minute, hour, day, week, month, quarter, year)
Calculate period start and end boundaries automatically
Parse natural language expressions like "next Monday", "3 days ago", "in 2 weeks"
Add or subtract intervals across any time period
Move forward/backward through configurable time periods

# Supported Natural Language
Relative days: "tomorrow", "yesterday"
Named days: "next Monday", "last Friday"
Intervals: "3 days ago", "in 2 weeks"
Months: "January", "December"
Standard date formats

# Use Cases
Building date range pickers
Scheduling and calendar applications
Report generation with period filtering
Natural language date input parsing
Time-based data analysis tools

Requirements

.NET Framework 4.5+ or .NET Core/5+
